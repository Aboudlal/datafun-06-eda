## Dataset
We used the `tips.csv` dataset, which contains restaurant bill and tip information such as:
- Total bill
- Tip
- Gender
- Smoker status
- Day of week
- Time (Lunch/Dinner)
- Party size

## Requirements
- Python 3
- pandas
- seaborn
- matplotlib
- jupyterlab

## Setup
```bash
# Create and activate a virtual environment
python -m venv .venv
